# Project North Star Brand Reference Library

## Purpose

This folder is the permanent reference library for Project North Star’s visual
identity.

Every reference asset placed here should be documented in this file with a short
description explaining why it exists and how future design work should use it.

Reference assets exist to communicate:

- emotion
- atmosphere
- typography
- composition
- luxury
- restraint
- craftsmanship

They should never be copied literally. They should influence the overall visual
language while allowing original design solutions. The objective is consistency
of brand identity, not duplication of imagery.

## How to document assets

Use this format for every future reference asset:

```markdown
### asset-file-name.ext

Purpose:
Short explanation of why this asset exists.

Reference for:

- atmosphere
- color relationships
- typography mood
- visual rhythm

This is inspiration, not a layout to duplicate.
```

## Documented reference assets

### tim-bennett-platform-ecosystem.png

Purpose:
This infographic clarifies the brand hierarchy and platform narrative: Tim
Bennett is the founder and source of the vision, Project North Star is the
guiding philosophy, and the experiences are expressions of that philosophy.

Reference for:

- brand hierarchy
- founder presence
- platform breadth
- relationship between philosophy and experiences
- leadership, purpose, craftsmanship, and trust
- the visitor's progression from Tim Bennett to the experiences he has created

This is an information and narrative reference, not a website design reference.
Do not copy its boxes, diagrams, icons, connectors, density, or layout. Use it
to understand why the platform should feel like Tim Bennett's digital
headquarters rather than a photographer's portfolio.

### mood-board-v1.png

Purpose:
This mood board represents the emotional and visual direction of Project North
Star.

Reference for:

- emotional tone
- atmosphere
- design language
- luxury
- color relationships
- typography mood
- composition
- visual rhythm
- craftsmanship
- aspiration

This is not a website mockup and should not be reproduced literally. Instead,
it establishes the atmosphere, design language, typography, color relationships,
and emotional tone that should influence future design decisions.

Why this asset was chosen:

The mood board captures the qualities visitors should experience when they
arrive at Project North Star:

- luxury
- confidence
- sophistication
- calm
- timelessness
- craftsmanship
- aspiration

The website should feel intentional and refined rather than trendy or
technology-driven.

Typography guidance:

The typography shown here is one of the strongest elements of the reference.
Important characteristics include:

- elegant editorial styling
- generous spacing
- timeless serif display typography
- understated sophistication
- premium magazine aesthetic

The goal is not to use these exact fonts. Use them as inspiration when selecting
typography that communicates refinement while maintaining excellent readability
across all devices.

Color guidance:

The visual palette communicates luxury through restraint. Key characteristics
include:

- deep charcoal rather than pure black
- warm bronze tones
- restrained gold accents
- warm neutrals
- controlled contrast
- soft highlights

Gold should remain an accent color rather than becoming the dominant visual
element.

Layout guidance:

The board demonstrates compositional ideas worth preserving:

- generous negative space
- strong visual hierarchy
- photography leading the experience
- elegant alignment
- balanced compositions
- restrained use of decorative elements

The interface should feel calm and confident rather than busy.

Overall atmosphere:

The most important quality of this reference is its atmosphere. It feels:

- cinematic
- editorial
- architectural
- intimate
- premium
- emotionally engaging

The objective is not to recreate these images or layouts. The objective is to
create a website that evokes the same emotional response.

Implementation guidance:

When future visual refinements are proposed, evaluate them against this
reference by asking:

- Does this increase the feeling of luxury?
- Does this feel timeless rather than trendy?
- Does the typography communicate refinement?
- Does the layout create calm through simplicity?
- Does the interface support the photography instead of competing with it?

If the answer is no, reconsider the design. This mood board should influence
the visual language of Project North Star while allowing the website to remain
original, accessible, responsive, and technically excellent.

### project-north-star-logo.png

Purpose:
This logo is the primary visual identity of Project North Star. It should be
treated as the highest-level brand reference within the project.

Reference for:

- overall visual language
- primary brand identity
- gold accent usage
- geometry
- spacing philosophy
- refinement
- simplicity
- balance
- disciplined composition
- quiet confidence

This asset exists to establish the overall design language of the application,
not merely the appearance of the logo itself.

Why this asset was chosen:

The logo represents the values of Project North Star:

- direction
- purpose
- craftsmanship
- precision
- confidence
- exploration
- timelessness

Every visual decision throughout the application should feel compatible with
these values.

Typography guidance:

The typography is:

- elegant
- timeless
- refined
- confident
- editorial

Future typography selections should evoke a similar feeling while remaining
optimized for web readability.

Color guidance:

The gold is intentional. It communicates value without becoming overwhelming.
Use gold as a premium accent rather than a dominant interface color. Dark
charcoal and warm neutral tones should remain the primary foundation.

Geometry guidance:

The logo is built on balance and symmetry. The interface should reflect that
same discipline through:

- alignment
- spacing
- proportion
- visual hierarchy
- consistency

Atmosphere guidance:

The logo communicates:

- luxury
- aspiration
- adventure
- craftsmanship
- quiet confidence

The application should evoke these qualities without becoming ornate or
decorative.

Implementation guidance:

The logo should influence the overall visual language of the website. It should
not become a repeated decorative motif.

Avoid repeating stars, circles, compass graphics, mountains, or other literal
elements from the logo throughout the interface.

Instead, let the logo inspire:

- typography
- spacing
- balance
- restrained use of gold
- premium presentation
- disciplined composition

The objective is that a visitor should feel the website belongs to the same
brand as this logo, even if the logo were temporarily removed from the page.

Relationship to the mood board:

The mood board establishes the emotional atmosphere. This logo establishes the
brand identity. Future visual refinements should satisfy both.

The ideal result is a website that feels as luxurious as the mood board while
remaining unmistakably Project North Star.

## Future workflow

Before beginning any significant visual design work, review:

1. `docs/brand/design-philosophy.md`

3. Every documented asset in this reference library

These documents collectively define the visual direction of Project North Star.

## Design authority

The existing implementation remains the architectural foundation of the project.

Future visual refinement should preserve:

- UX
- accessibility
- responsive behavior
- component architecture
- performance

The objective is to elevate the visual language rather than redesign the
application’s structure. Treat the existing implementation as architecturally
complete unless explicitly instructed otherwise.
