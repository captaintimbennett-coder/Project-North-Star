# Project North Star Foundation Pack

These files are the official source of truth for Project North Star as of
June 30, 2026. They replace all previous creative direction.

## Governing files

1. `the-north-star-constitution-v1.0.docx` governs mission, values, founder
   philosophy, user experience, brand personality, builder principles, design
   non-negotiables, technical philosophy, and the North Star Promise.
2. `design-principles.md` is the permanent creative philosophy governing every
   future page, feature, component, and user experience.
3. `north-star-visual-reference-v1.0.png` is the approved homepage and Hero
   design reference. Faithful implementation is required.
4. `founder-portrait-tim-bennett.png` is the approved Version 1.0 production
   portrait. A future dedicated portrait may replace it without blocking the
   current implementation.
5. `foundation-review-implementation-brief.docx` defines the review,
   implementation rules, technical expectations, and success criteria.
6. `lone-star-retreat-code-of-conduct.md` governs professional standards,
   participant culture, safety, consent, and conduct for Lone Star Retreat.
7. `scheduling-and-booking-rules.md` governs event-specific
   availability, booking, calendar visibility, contact sharing, financial
   boundaries, and administrator scheduling authority for Lone Star Retreat.

## Approved implementation clarifications

- Use the supplied founder portrait for Version 1.0 and do not delay for a new
  portrait.
- Use the closest available premium editorial serif. Structural fidelity takes
  priority over final typography selection.
- Complete and review the Hero before proceeding to the next homepage section.

Older documents remain useful for technical context only where they do not
conflict with this Foundation Pack.
